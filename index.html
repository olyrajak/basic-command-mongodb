Install MongoDb In Ubuntu

Step 1 : wget -qO - https://www.mongodb.org/static/pgp/server-5.0.asc | sudo apt-key add -
step 2 : echo "deb [ arch=amd64,arm64 ] https://repo.mongodb.org/apt/ubuntu focal/mongodb-org/5.0 multiverse" | sudo tee /etc/apt/sources.list.d/mongodb-org-5.0.list
step 3 : sudo apt update
step 4 : sudo apt install -y mongodb-org




Some Basic Command

Create a new Database : sample database_name

Create New Collection : db.createCollection("collecton_name")

create a capped collection And with maximum of 3 documents :db.createCollection("collecton_name", { capped: true, size: 3})

insert : db.collecton_name.insert({state: "State1",price:600})

Check Collection Status : db.collecton_name.stats()

Delete Collection: db.collecton_name.drop()

InsertMany: db.collecton_name.insertMany([{}])

Find Perticular: db.collecton_name.find({state:"State1"})

Find And Update: db.collecton_name.update({state:"State1"},{$set:{state:State2}})

Find And delete: db.collecton_name.remove({state:"State1"})

Delete Database : db.dropDatabase()

drop all documents from `collecton_name` collection. : db.collecton_name.remove({})

drop `collecton_name` collection. : db.collecton_name.drop()

Exact Match: db.collecton_name.findOne({state:"State1"})

Exact Null: db.collecton_name.findOne({state:null})

Does Not Exist: db.collecton_name.find({state:{exists:false}})

greater than: db.collecton_name.find({state:{$gt:500}})

greater than Equal to: db.collecton_name.find({state:{$gte:500}})

less than: db.collecton_name.find({state:{$lt:700}})

less than Equal To: db.collecton_name.find({state:{$lte:700}})

Find all users whose name includes 'ri' in their name: db.collection_name.find({ name: /ri/i })

Find all users who play either 'football' or 'cricket': db.collection_name.find({sports: {$in : ['cricket', 'football']}})

rename details field to description from all articles in articles collection. : db.articles.updateMany({}, {$rename: {"details": "description"}}, {multi: true})

Import Json :mongoimport:jsonImporte='generated.json'
